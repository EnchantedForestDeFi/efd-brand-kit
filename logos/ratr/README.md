# RATR Logos (Native Ratatoskr)

Logo family for **Ratatoskr (RATR)** — the native PoW + masternode chain. Mainnet launches 2026-06-01 00:00 UTC.

**Style:** Illuminated-manuscript / engraved-woodcut. Red squirrel on a mossy branch carrying a parchment scroll with horseshoe seal, framed by Celtic trefoil corners on aged parchment. Locked 2026-05-26.

The wRATR family (`../wratr/`) is derived from this base by adding an Alephium round-logo seal hanging on the scroll cord below the horseshoe seal. RATR and wRATR are visually a matched pair: same scroll, native RATR has one seal, wRATR has two (double-sealed for the wrapping).

## Spec reference

Production spec authoritative in internal `efd_brand_asset_spec.md` Section 3. Public summary:

- Squirrel-as-messenger silhouette (Ratatoskr the messenger of Yggdrasil)
- Tree / bridge motif optional in extended marks
- Always rendered with brand-primary as silhouette body; brand-accent for tail-flash, eye, or accent stroke
- Wordmark uses Cinzel SemiBold
