## Permissions discipline

- **Before adding a new side-coin logo to this folder, verify the project's logo license / fair-use posture.**
- If the project has a public brand kit / press resources page, link it here as the source.
- If the project is small / informal, keep the logo only if there's clear no-objection from the project maintainers.
- If asked to remove a side-coin logo by the project maintainers, do so promptly.
