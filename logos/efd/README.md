# EFD House Brand Logos

Logo family for **Enchanted Forest DeFi (EFD)** — the parent / house brand that publishes RATR + operates the wRATR bridge + (future) hosts NFT marketplace and additional products.

---

## Files (when present)

| File | Use case |
|---|---|
| `efd-mark-1024.png` | Master raster, transparent BG |
| `efd-mark-512.png` | High-res web / press |
| `efd-mark-256.png` | Standard web / social use |
| `efd-mark.svg` | Vector master |
| `efd-wordmark.png` | Text wordmark (no symbol) |

---

## Brand palette

- **Secondary (signature):** `#2E5339` (forest green — the "enchanted forest" tie-in)
- **Accent:** `#D7882F` (ember orange — shared with RATR through-line)
- **Light background:** `#EDE2C5` (parchment)

See [`/palette/brand-colors.md`](../../palette/brand-colors.md).

---

## Use guidelines

- Use the EFD mark for **parent-brand** contexts (press, partnership, multi-product references)
- Use the RATR mark for **chain-specific** contexts (CEX listings, miner-facing material)
- Both marks may co-exist (e.g., "EFD presents RATR" in a launch banner)
