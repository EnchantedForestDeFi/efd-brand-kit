# Hero Images

Large-format hero images for press kits, BitcoinTalk thread headers, social media, and EFD website hero sections.

All are ~2-3 MB PNGs at 2048+ pixels — too large for icon use; intended for hero positions where the artwork is the focal element.

| File | Description | Recommended use |
|---|---|---|
| `hero-canonical-mark.png` | The canonical Ratatoskr (squirrel-on-branch) mark at hero scale | Default hero position on EFD website, RATR press kit cover |
| `hero-color-manuscript.png` | The same mark rendered in full illuminated-manuscript color treatment | BitcoinTalk thread header, social media hero card |
| `hero-narrative-map.png` | Wider composition showing Yggdrasil + Nine Worlds context | Lore page hero, whitepaper cover, "What is EFD?" presentations |

## Source

Created during the May 2026 brand identity work. Originals lived in `ratatoskr/doc/brand/heroes/` in the chain repo; mirrored here as the canonical distribution location.
