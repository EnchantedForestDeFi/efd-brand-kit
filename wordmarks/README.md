# EFD Wordmarks (Text-Only Marks)

Text-only treatments of EFD brand names — no symbol. Use when:

- The visual symbol is shown elsewhere in the layout
- The medium can't render raster/vector well (very-low-resolution contexts)
- The layout has space for typography but not iconography
- Print contexts where the mark is too small to read

---

## Files (when present)

| File | Use case |
|---|---|
| `ratr-wordmark.svg` | "RATR" / "Ratatoskr" text mark |
| `efd-wordmark.svg` | "EFD" / "Enchanted Forest DeFi" text mark |
| `ratatoskr-full-wordmark.svg` | Long-form "Ratatoskr" wordmark |

---

## Typography

Wordmarks use **Cinzel SemiBold** at letterforms tuned for the mark (not raw text — letter spacing + weight adjusted for display).

See [`/typography/fonts.md`](../typography/fonts.md) for full type system.
