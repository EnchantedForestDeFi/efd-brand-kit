# Contributing to the EFD Brand Kit

This repository is the **canonical** public home for EFD brand assets. Maintained primarily by EFD founders and designated brand custodians.

---

## What kinds of contributions are welcome?

- **Typo fixes** in any markdown file
- **Missing standard sizes** (e.g., we have 256px but not 128px for a logo)
- **Improved file optimization** (smaller PNG without visible quality loss)
- **Translation** of palette/typography docs to other languages
- **Bug reports** for broken links, missing files, or malformed assets

---

## What is NOT welcome via PR

- **Substantive logo redesigns** — these are brand decisions, not community PRs. Open an issue first to discuss.
- **New character artwork** — pantheon characters have canonical visual specs (see `characters/`). Adding or substantially redesigning a character is a brand decision.
- **Changes to palette HEX values** — locked per canonical brand spec.
- **Changes to license terms** — CC-BY 4.0 is locked.

---

## How to submit

1. **Fork** this repo
2. **Branch** from `main` (e.g., `fix/typo-readme` or `add/ratr-mark-32px`)
3. **Commit** your change with a clear message
4. **Open a PR** describing what changed and why
5. Maintainer reviews and merges if appropriate
